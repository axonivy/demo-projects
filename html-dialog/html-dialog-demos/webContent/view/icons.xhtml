Primefaces/Icon Showcase Icon Showcase

With the themes you have several possibilities to insert an icon into your page.

  • Tabler Icons
  • PrimeIcons
  • Streamline Icons Collection (Deprecated)
  • FontAwesome 6.1.0 (Deprecated)

You can use them for example as follows:

Tabler Icons

<i class="ti ti-book"></i> <p:commandButton icon="ti ti-settings" value="Settings" />

Tabler Icons Filled

<i class="tif tif-book"></i> <p:commandButton icon="tif tif-settings" value="Settings" />

PrimeIcons

<i class="pi pi-home"></i> <p:commandButton icon="pi pi-cog" value="Settings" />

Streamline Icons (Deprecated)

<i class="si si-house-1"></i> <p:commandButton icon="si si-cog" value="Settings" />

FontAwesome (Deprecated)

<i class="fa-solid fa-house"></i> <p:commandButton icon="fa-solid fa-gear" value="Settings" />

Please us a modern browser like Chrome, Firefox or Edge to browse the Streamline Icon Collection!
